旅游之家数据库表

1.导游表 pmw_guide

id
name          导游姓名(varchar)
sex           导游性别(int)
card          导游证(varchar)
cardnumber    导游证号(varchar)
tel           导游电话(varchar)
images        导游头像(varchar)默认第一次拉取微信头像
account       账号(varchar)
password      密码(varchar)
content       简介(选填)(text)
pics          导游相册 (选填)(text)
regtime       注册时间 (int)
ymdtime       注册年月日（date）
regip         注册ip（varchar)
getcity       注册获取当前得城市（varchar)
checkinfo     注册审核状态（0待审核1审核通过2审核失败）
openid        用户通过小程序进入得到得openid
formid        过户发送模板消息的时候用到的formid

3.旅行社注册未成功发送模板消息  pmw_agency_message

id
tp             账户类型(varchar)
name           企业名称(varchar)
tel            联系电话(varchar)
state          审核状态(varchar)
content        失败原因(content)
applytime      申请时间(datetime)
sendtime       审核时间(datetime)


4.旅行社发布旅游行程 pmw_travel

id
title           行程标题
starttime       开始时间
endtime         结束时间
days            行程时长
num             团队人数
origin          客源地
content         添加行程   用json字符串表示，中间用|隔开
money           导游费用
other           其他备注
aid             旅行社id
gid             导游id（默认为空，如果导游接了这单，则显示导游的id）
state           发布状态（0，待预约，1:待确认 2:已确认（已完成），如果导游没有确认，
                则系统默认在出发的前一天进行确认处理3:已取消，在已完成里面不能取消）
comment_state   旅行社给导游的评价  0，默认未评价，1已经评价

addtime        评价时间

8.系统发送模板消息表  pmw_message

id
type           会员类型 (agency  guide)
messagetype    消息类型 (system 系统消息  template 模板消息)
templatetype   模板消息分类(cancel  取消行程   appointment预约行程, reg 注册成功 如果不是模板消息的话则为空)
content        消息内容
title          消息标题
mid            会员id(旅行社的id 或者导游的id)
faxtime        发布时间
state          状态(0,未读，1已读)
